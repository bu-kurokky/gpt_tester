  • Chat GPT for API α

    OpenAIのChatGPTをAPI Keyをサーバーサイドを使わずにJavascriptのみで直接叩いた結果を返すものです。

    使い方&ご注意

    • あくまで「APIkeyあるのにクライアント作るの面倒だー」って人向けです。
    • 必ず一番下のOpenAIのAPIを入力してから使用してください。(入力しないと使えません)
    • 一部のデータはLocalStorageに平文で突っ込んでるので他人が使う環境では十分注意してください。
    • チャット結果が全て返ってきてからmarkdownをHTML変換しているので途中はあくまで雰囲気です。
    • スーパー突貫(思いついてから作るまで2時間)で作ってるので色々不備があります。
    • チャットのやりとりをスレッド？としてまとめて問い合わせに行く仕様にしてるので文字数オーバー、課金とかやばくなる可能性があるので気をつけてください。
    • ↑一応対策として自分の中で話題が完全に変わったら、左側にある「新しい質問をする」をクリックしてから問い合わせしてください。
